
# Gitinder 

A small application that utilizes Github API to fetch Github users and display their profiles in detail.




## Authors

- [@furkancodes](https://www.github.com/furkancodes)


## Demo

https://gitinder.vercel.app/

## Installation

Install with npm

```bash
  npm install 
  cd ProjectName
```
